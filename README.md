# MoleMash
**Version 0.1**
* Touch sensor
* Hit Score
* Miss Score
* Randomized location of a sprite
* Reset score button
# How to import
**Mac**
* Download the zip by pressing code and pressing download zip
* unzip it in your files by double pressing the zip
* Go to Mit app inventor and press the button next to the new project file (It'll be on the right to it)
* Press choose file and look for MoleMash-Main
* Once you pressed MoleMash-Main press the Source file and open it
